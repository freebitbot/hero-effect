# Пакети для виключення (Поверхневий аналіз)

Цей документ містить поверхневий аналіз пакетів, які не будуть включені в нову імплементацію: Datastores, Desktop та Platform.

---

## 1. DATASTORE (apps/datastore/)

### Огляд

Datastore - це система для створення деплоябельних scraping API. Дозволяє створювати платні API сервіси з даними, отриманими через Hero скрапінг.

### Пакети групи

#### @ulixee/datastore
**Призначення:** Головний пакет для створення Datastore - деплоябельного scraping API.

**Бізнес-функція:**
- Створення API ендпоінтів з веб-скрапінгу
- Підтримка SQL для запитів до даних
- Інтеграція з Argon Protocol для платежів (криптовалюта)
- Збереження даних у SQLite

**Ключові експорти:**
- `lib/*` - основна логіка
- `cli/*` - CLI інструменти
- `storage-engines/*` - движки зберігання
- `payments/*` - інтеграція платежів
- `connections/*` - підключення до Cloud

**Залежності:**
- `@argonprotocol/localchain`, `@argonprotocol/mainchain` - блокчейн для платежів
- `better-sqlite3` - локальна база даних
- `@ulixee/sql-*` - SQL парсинг та виконання
- `@ulixee/schema` - валідація даних

---

#### @ulixee/datastore-core
**Призначення:** Ядро Datastore з логікою виконання та управління сесіями.

**Бізнес-функція:**
- Запуск та виконання Datastore скриптів
- Управління сесіями та базами даних
- Пакування Datastore для деплою

**Залежності:**
- `tar` - архівація для деплою
- `nanoid` - генерація ID
- Всі основні platform пакети

---

#### @ulixee/databroker
**Призначення:** CLI інструмент для хостингу та управління Datastores.

**Бізнес-функція:**
- Запуск Datastore сервера
- Адміністрування деплоїв
- Монетизація даних через Argon Protocol

**Залежності:**
- `@ulixee/cloud` - хмарна інфраструктура
- `@argonprotocol/*` - блокчейн інтеграція

---

#### @ulixee/datastore-plugins-hero
**Призначення:** Hero плагін для Datastore.

**Бізнес-функція:** Використання Hero скрапінгу всередині Datastore.

---

#### @ulixee/datastore-plugins-hero-core
**Призначення:** Ядро Hero плагіна для Datastore.

---

#### @ulixee/datastore-plugins-puppeteer
**Призначення:** Puppeteer плагін для Datastore.

**Бізнес-функція:** Альтернатива Hero - пряме використання Puppeteer.

---

#### @ulixee/datastore-plugins-puppeteer-core
**Призначення:** Ядро Puppeteer плагіна.

---

#### @ulixee/datastore-packager
**Призначення:** Упаковка Datastore в деплоябельний формат.

**Бізнес-функція:** Створення xarg файлів для деплою на Cloud.

---

#### @ulixee/datastore-docpage
**Призначення:** Автоматична генерація документації для Datastore API.

**Бізнес-функція:** Створення веб-сторінок з документацією API.

---

### Висновок по Datastore

Datastore - це **система монетизації скрапінгу**. Вона перетворює Hero скрипти на платні API з:
- Автоматичною документацією
- Оплатою через криптовалюту (Argon Protocol)
- SQL інтерфейсом для запитів
- Хмарним деплоєм

**Ця функціональність не потрібна для базової бібліотеки скрапінгу.**

---

## 2. DESKTOP (apps/desktop/)

### Огляд

Desktop - це GUI оболонка для Hero та Datastore для десктопного використання.

### Пакети групи

#### @ulixee/desktop-interfaces
**Призначення:** TypeScript інтерфейси для desktop додатку.

**Залежності:**
- `@ulixee/hero-interfaces`
- `@ulixee/platform-specification`
- `@ulixee/unblocked-specification`

---

#### @ulixee/desktop-core
**Призначення:** Ядро desktop додатку.

**Бізнес-функція:**
- GUI для Hero скрапінгу
- Time travel дебагінг (відтворення сесій)
- Візуалізація результатів скрапінгу
- Пошук у DOM структурі

**Ключові залежності:**
- `fuse.js` - fuzzy пошук
- `ws` - WebSocket з'єднання
- `@ulixee/hero-timetravel` - відтворення сесій

---

### Висновок по Desktop

Desktop - це **GUI додаток** для візуальної роботи з Hero:
- Time travel дебагінг
- Візуальний інспектор
- Пошук по DOM

**Ця функціональність корисна для розробки, але не критична для бібліотеки.**

---

## 3. PLATFORM (apps/platform/)

### Огляд

Platform - це інфраструктура для хмарного деплою та монетизації.

### Пакети групи

#### @ulixee/cloud
**Призначення:** Хмарний сервер для хостингу Datastores та Hero скриптів.

**Опис (з package.json):**
> "The Ulixee Cloud is an out-of-the box tool to host and deploy Ulixee Datastores and Hero scripts. It allows you to easily create a paid API service."

**Бізнес-функція:**
- Хостинг Datastore API
- Управління сесіями в хмарі
- Монетизація даних

---

#### @ulixee/cli
**Призначення:** CLI інструмент платформи.

**Залежності:**
- `commander` - парсинг аргументів

---

#### @ulixee/schema
**Призначення:** Система схем для типізації даних.

**Опис (з package.json):**
> "A basic schema utility with static constraints to support secure type checking."

**Залежності:**
- `moment` - робота з датами

---

#### @ulixee/sql-ast
**Призначення:** AST (Abstract Syntax Tree) парсер для SQL.

**Залежності:**
- `moo` - лексер
- `nearley` - парсер

---

#### @ulixee/sql-engine
**Призначення:** Движок виконання SQL запитів.

**Бізнес-функція:** SQL інтерфейс для запитів до даних Datastore.

---

#### @ulixee/client
**Призначення:** Клієнт для підключення до Cloud.

**Бізнес-функція:**
- Підключення до віддалених Datastores
- Оплата запитів через Argon Protocol

---

#### @ulixee/platform-utils
**Призначення:** Спільні утиліти платформи.

---

#### @ulixee/platform-specification
**Призначення:** Специфікації протоколів платформи.

**Залежності:**
- `@argonprotocol/localchain` - блокчейн
- `zod` - валідація схем

---

### Висновок по Platform

Platform - це **інфраструктура для монетизації**:
- Хмарний хостинг
- SQL інтерфейс для даних
- Платежі через криптовалюту
- Клієнт-серверна архітектура

**Ця функціональність не потрібна для базової бібліотеки скрапінгу.**

---

## Загальний висновок

Пакети Datastore, Desktop та Platform призначені для:

1. **Монетизації** - створення платних API з даних скрапінгу
2. **Хмарного деплою** - хостинг сервісів
3. **GUI** - візуальні інструменти для розробників
4. **SQL інтерфейсу** - запити до даних

Для оптимізованої імплементації ці функції можна виключити, залишивши тільки ядро браузерної автоматизації та скрапінгу.

**Залежності, які можна виключити:**
- `@argonprotocol/*` - блокчейн
- `better-sqlite3` - база даних
- `tar` - архівація
- `fuse.js` - пошук
- SQL-пов'язані пакети
